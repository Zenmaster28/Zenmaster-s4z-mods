Instructions for using the Virtual goal pacer.
  • You will need to use the Sauce for Strava browser extension.
  • Open the Strava activity that you wish to follow, it can be one of your own or someone else's (as long as you have access to it).
  • Click to the Analysis Tab along the left.
  • Scroll down to the graphs at the bottom.
  • If you want to use the entire activity, click the FIT link from the Data section of the Sauce for Strava bar.
  • If you want to use only part of the activity, click and drag on the graph to highlight the section you want to use and then click the FIT link from the Data section.
  • If prompted for an Activity Start time, ensure one is filled out (it should be for a Zwift activity) and then click Export.
  • Save the file in a location you can remember as you will need to reopen it in the goal pacer window.
  • In the Virtual goal pacer window, right click and then click the open button to import data.
  • Browse to the file you downloaded earlier and click Open.
  • Start your ride, the window will indicate when you are within 1000m (distance can be changed in the settings) of the start.
  • Once you enter the route/segment/climb/whatever, it will indicate progress as to whether you are ahead/on pace/behind the goal. It will also show distance and goal time information. You can set it minimalist to only show the ahead/behind in the settings.
